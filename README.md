# Chocolate-Sales-Intelligence-Dashboard-Analysis-Segmentation-and-Revenue-Forecasting
End-to-End Chocolate Sales Analytics Project using Python, SQL, Power BI and Machine Learning. Includes customer segmentation (KMeans), revenue forecasting (Prophet), advanced SQL analysis, and interactive dashboards to derive business insights and drive decision-making.



# 🍫 Chocolate Sales Intelligence Dashboard: Analysis, Segmentation and Revenue Forecasting

---

## 📌 Project Overview

This project delivers a complete end-to-end data analytics solution for chocolate sales data, transforming raw transactional data into actionable business insights and future revenue predictions.

It combines **data cleaning, SQL-based analysis, exploratory data analysis (EDA), customer segmentation, time-series forecasting, and interactive dashboarding** to support data-driven decision-making.

---

## 🎯 Business Problem

Businesses often struggle to:

* Identify high-performing products and categories
* Understand customer purchasing behavior
* Evaluate the true impact of discounts on profitability
* Forecast future sales for planning and inventory

This project addresses these challenges using data analytics and machine learning techniques.

---

## 📊 Dataset Information

* **Source:** Sales dataset (retail chocolate transactions)
* **Records:** ~200,000+ rows
* **Features:** 30+ columns
* **Time Period:** 2023 – 2025
* **Key Metrics:** Revenue, Profit, Cost, Quantity, Discount

---

## 🧹 Data Cleaning & Preprocessing

* Handled missing values (e.g., `age_group → "Unknown"`)
* Converted date columns to proper datetime format
* Removed inconsistencies in categorical data
* Feature engineering:

  * Month, Year, Week, Day extraction
  * Revenue, Profit validation
* Standardized numerical features for clustering

---

## ⚙️ Methodology

### 1. Data Preparation

* Imported dataset into SQL and Python
* Cleaned and structured data for analysis

### 2. SQL Analysis

* Aggregations and grouping
* Window functions (RANK, ROW_NUMBER)
* Business queries (top stores, revenue trends)

### 3. Exploratory Data Analysis (EDA)

* Distribution analysis (histograms)
* Relationship analysis (scatter plots)
* Correlation heatmaps

### 4. Customer Segmentation

* Applied **KMeans Clustering**
* Features used: Age, Quantity, Revenue
* Optimal clusters selected using **Elbow Method**

### 5. Forecasting

* Built time-series model using **Prophet**
* Generated future revenue predictions
* Captured **trend + seasonality patterns**

### 6. Dashboard Development

* Built interactive dashboards in Power BI
* Multi-page reporting:

  * Overview
  * Product Analysis
  * Customer Analysis
  * Location Analysis
  * Advanced Insights

---

## 📈 Forecast Evaluation

Model performance evaluated using:

* RMSE (Root Mean Squared Error)
* MAE (Mean Absolute Error)
* MAPE (Mean Absolute Percentage Error)
* Coverage

These metrics ensure reliability of predictions.

---

## 🔍 Key Insights

* High revenue is generated by specific premium categories
* Customers aged **50+ contribute significantly to revenue**
* Discounts increase sales volume but reduce profit margins
* Certain locations outperform others consistently
* Loyal customers show higher repeat purchase behavior
